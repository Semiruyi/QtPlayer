# 本地视频播放器

这是一个本地视频播放软件，



## 使用方法

1. 打开应用程序。
2. 点击界面上的“Add”按钮，选择一个有视频的文件夹。
3. 即可即刻探索（误

## 注意事项

- 播放列表默认按文件名字典序从小到大排序，（2会在12后面，02会在12前面）
- 支持的视频格式暂时只支持并测试了 MP4,  MKV 格式。

## 依赖

- release已打包依赖，双击appQtPlayer.exe即可运行
